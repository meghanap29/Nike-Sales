#  Nike Sales Insights Dashboard

This project presents an end-to-end data analysis of Nike's sales dataset — from raw data cleaning in Python to interactive dashboard visualization in Tableau.

##  Project Overview

This repository includes:
-  Data cleaning script written in Python
-  A Tableau dashboard visualizing sales, profit, and product performance
-  Both uncleaned and cleaned versions of the dataset
-  Image snapshot of the final dashboard

##  Dataset

📂 **Source**: [Nike Sales (Uncleaned) Dataset – Kaggle](https://www.kaggle.com/datasets/nayakganesh007/nike-sales-uncleaned-dataset)

- `nike_sales_uncleaned.csv` → Original dataset
- `Nike_Sales_SoftCleaned_Final.csv` → Cleaned version used in Tableau

---

##  Data Cleaning

The dataset was cleaned using a Python script to:
- Remove duplicates
- Standardize region names (e.g., "Hyd" → "Hyderabad")
- Handle null values
- Format dates for analysis

 `clean_data.py` contains the complete code with inline comments.
